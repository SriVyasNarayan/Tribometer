# Tribometer
Constructed a Stick-Slip Tribometer and tested its accuracy.
A stick-slip tribometer was designed and constructed using meticulous 3-D printing techniques.
The material (Metal) was made to be in contact with the sliding surface (sandpaper) with the help of a normal laod on top of the sliding block. This sliding block was attached to an extension spring and this spring was pulled.
The extension of the spring was measured and used to calculate the friction coefficient of the respective material with the sandpaper used.
These results were compared to the pre-existing data and the accuracy of the tribometer was determined.
